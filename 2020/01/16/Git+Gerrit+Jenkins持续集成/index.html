5. 使用教程

  • 访问地址 http://IP:8170

  • 安装git

  • 通过ssh-keygen -t rsa -C "邮箱"生成公私钥，并把公钥复制到

  • 克隆代码到本地，BROWSE->点击对应的项目->点击Clone with commit-msg hook下面克隆的链接，别选其他链接进行克隆

  • 修改个人账户密码

    登录服务器，输入htpasswd /home/gerrit/gerrit.password xxx，这里的xxx是个人gerrit账号名，也就是自己的姓名，再输入两次密码即可成功修改
