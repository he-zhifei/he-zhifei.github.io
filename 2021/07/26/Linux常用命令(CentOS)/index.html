Linux常用命令(CentOS)

发表于 2021-07-26 更新于 2024-10-19 分类于 Linux ， CentOS 阅读次数： 本文字数： 8.4k 阅读时长 ≈ 31 分钟

1. 基础命令

提示：用法中的[]表示可选参数，<>表示必选参数。

1.1 shutdown

1.2 –help参数

用法：<命令> –help 查看命令的相关用法

1.3 man

用法：man <命令> 查看命令的用户手册，q-退出，空格-下一页，b-上一页

2. 目录命令

2.1 cd

用法：cd [目录] 切换到指定目录，可以是相对路径，也可以是绝对路径。

2.2 ls

用法：ls [参数]… [目录或文件]… 查看目录或文件

2.3 mkdir

用法：mkdir [选项]… 目录… 创建目录

2.4 rm

用法：rm [选项]… 文件… 删除文件或文件夹

2.5 mv

用法：mv [选项]… [-T] 源文件 目标文件 （移动并）重命名
　或：mv [选项]… 源文件… 目录 源文件移动到指定目录下
　或：mv [选项]… -t 目录 源文件… 源文件移动到指定目录下

2.6 cp

用法：cp [选项]… [-T] 源文件 目标文件 文件复制
　或：cp [选项]… 源文件… 目录
　或：cp [选项]… -t 目录 源文件…

2.7 find

2.8 pwd

用法：pwd 查看当前工作目录

3. 文件命令

3.1 touch

用法：touch [选项]… 文件… 创建文件（若已存在，则只更新文件的修改时间）

3.2 vi(vim)

用法：vi <文件> 编辑文件（若不存在，则在保存退出后创建），有3种模式：命令模式（默认）、插入模式、末行模式。

若命令缺失，先安装：yum -y install vim

在命令模式下：

  • ctrl + f 下一页，ctrl + b 上一页；
  • 0 光标移动到行首字符，$ 光标移动到行尾字符；
  • G 光标移动到最后一行，gg 光标移动到首行；
  • dd 删除光标所在行；
  • yy 复制光标所在行，p 在光标所在行的下一行粘贴，P 在光标所在行的上一行粘贴；
  • u 撤销修改，vi只能撤销上一次，vim可无限次撤销；

在命令模式下，输入以下的任一个字符进入插入模式：

  • i 从光标位置插入；
  • I 从光标所在行的第一个非空白字符插入；
  • o 从光标所在行的下一行插入；
  • O 从光标所在行的上一行插入；
  • a 从光标位置下一个字符插入；
  • A 从光标所在行的行尾插入；

在命令模式下，输入以下的任一个字符进入末行模式：

  • / 从光标位置往下查找字符，/abc 向下查找abc（n 下一个，N 上一个）；
  • ? 从光标位置往上查找字符，/abc 向下查找abc（n 下一个，N 上一个）；
  • : 可以替换字符，:1,$s/abc/ABC/gc 表示从第一行到最后一行查找并替换，把abc替换成ABC，并且每次替换前需要确认（如果没加参数c则不需要确认），$代表末行，可以指定具体行；:wq 表示保存并退出；:q! 表示不保存退出；

注：按 ESC 可从插入模式或末行模式退回到命令模式。

3.3 cat

用法：cat [选项]… [文件]… 查看文件

3.4 less

用法：less [选项] [文件]… 查看文件，回车-下一行，空格-下一页，b-上一页，q-退出，/往后检索，?往前检索，G最后一页，gg首页。

3.5 more

用法：more [选项] 文件… 查看文件，回车-下一行，空格-下一页，b-上一页，q-退出，到达最后一页会自动退出。

3.6 head

用法：head [选项]… [文件]… 从头查看文件

3.7 tail

用法：tail [选项]… [文件]… 从尾查看文件

4. 权限命令

权限说明：通过 ls -l 可查看文件或目录的权限。比如 drwxr-xr-x 第1位为d说明是目录（-为文件），后续每3位为一组，每组各位分别代表（r可读；w可写；x可执行，省缺”-“代表无该权限），第一组是所有者的权限，第二组是属组（所有者所属的组），第三组是其它用户的权限。每组权限分别用二进制表示，这个权限为755。只有所有者和root才能修改其文件的权限。

6.24 iptables(centos6)

  • -A（append）和-I（insert）区别，-A是追加在规则后方，-I是在最前面插入，如果使用追加的方式，可能导致追加的端口在REJECT的后方，导致端口没有开放成功。因此添加拒绝策略最好用-A参数，添加开放规则最好用-I。
  • 如果想要防火墙重启后还生效，需要执行 /etc/init.d/iptables save 进行保存防火墙策略
