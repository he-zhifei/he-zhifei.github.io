常用linux命令(centos)

发表于 2021-07-26 更新于 2024-02-19 分类于 linux ， centos 阅读次数： 本文字数： 6.7k 阅读时长 ≈ 24 分钟
1. ls
  • ls [OPTION]… [FILE]…
2. cd
  • 切换到指定目录
  • 切换到上一次的目录，只能两个目录之间来回切换
  • 切换到当前用户根目录
3. pwd
  • 查看当前工作目录
4. cat
  • 查看文件
5. more
  • 分页查看大文件，回车下一行，空格下一页，b上一页，q退出，到达最后一行后自动退出
6. less
  • 查看大文件，并显示行号(-n)，回车下一行，空格下一页，b上一页，q退出，/检索内容（加?往前匹配，G最下面，gg顶部），到达最后一行后不会自动退出
7. head
  • 从头查看文件
8. tail
  • 从尾查看文件
9. mkdir
  • 创建文件夹
10. rmdir
  • 删除空目录
11. touch
  • 创建一个空文件，当文件存在时，则只更新文件的创建时间
12. mv
  • 文件（夹）移动或重命名
13. cp
  • 文件复制，复制目录，通常-rf连用
14. rm
  • 永久删除文件或文件夹，-i会有提示，-f(force)提示默认选择y，-r(recursive)递归删除文件夹以及其下的所有内容
15. vim
  • 编辑文件，i进入插入模式，esc再回到一般模式，在一般模式按:进入底行模式，按q!退出不保存，wq退出并保存
16. 管道命令

  • 管道命令|，正则搜索命令grep（global search regular expression(RE) and print out the line,全面搜索正则表达式并把行打印出来），

    例如： 列出当前目录的详细信息并查找含有COND（忽略大小写-i）的文件或者目录

17. grep
  • 在每个FILE或是标准输入中查找PATTERN（egrep:支持正则表达式）
18. awk
19. sed
20. whereis
  • 通常用来查找某条命令的位置（$PATH范围内），例如：whereis docker
21. find
  • 在/root以及其任意目录下，查找以.txt结尾的文件或目录
  • 在dist目录下查找名为aaa的目录，-atime -x:在最近x天内使用过的文件，-mtime +y:在最近y天之前修改过的文件
22. locate(centos7需要手动安装)
  • 快速定位到所有的doc文件，root可以通过updatedb来更新文件名数据库
23. tar
  • 解压，例如把当前目录下的aaa.tar.gz解压到目录bbb中，加-C指定路径，z(使用gzip), c(create), v(verbose), f(file), x(extract)
  • 压缩，把aaa/目录压缩成aaa.tar.gz包
24. 查看ip
  • ifconfig（若命令缺失，yum -y install net-tools）
  • ip -a address
25. 进程相关命令
26. 内存相关命令
27. 权限相关命令
  • 只有属主和root才能修改文件的权限
28. 远程操作命令
  • ssh远程登录
  • scp远程复制
  • rsync远程同步（执行yum -y install rsync进行安装），常用avzh参数
29. systemctl
30. firewalld
  • 服务相关
  • firewall-cmd命令，端口可为tcp或udp
30. iptables
  • -A（append）和-I（insert）区别，-A是追加在规则后方，-I是在最前面插入，如果使用追加的方式，可能导致追加的端口在REJECT的后方，导致端口没有开放成功。因此添加拒绝策略最好用-A参数，添加开放规则最好用-I。
  • 如果想要防火墙重启后还生效，需要执行 /etc/init.d/iptables save 进行保存防火墙策略
